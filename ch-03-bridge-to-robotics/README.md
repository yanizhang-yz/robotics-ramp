# Chapter 3 — Bridge to robotics

> ⬜ Not started. Scoped in the [ROADMAP](../ROADMAP.md) learning track.

Where the learning track meets robot policies:

- **Behavior cloning** — supervised learning from demonstrations.
- **Diffusion Policy** — action prediction as denoising. The PushT experiment in the
  `robotics-experiments` repo is the hands-on counterpart to the theory here.
- **ACT** (Action Chunking Transformer) — the SO-101 starter policy; structurally the
  [Chapter 1](../ch-01-transformer/) transformer with image/state tokens and a
  next-*action* objective.

Build-first exercises land here; the hardware milestones that use them (M1+) live in
`robotics-experiments`.
